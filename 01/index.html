RO.ME "3 DREAMS OF BLACK"

RO.ME is an interactive website based on the film "3 Dreams of Black." It was made with WebGL, which allows 3D graphics to run in a web browser. The project was launched in 2011 (12 years ago), which is a revolutionary work for that time period, as there were not many interactive and visually stunning works that could be run on a website, using the power from the graphics card in the computer. The demo is an experience filled with great 3D visuals, audio, and you can move the camera as if you were in the demo. Currently, we can see a whole environment where enthusiasts are free to contribute to open-source projects and create more interactive and fascinating websites using WebGL and 3D graphics.

Technology: 3D, WebGL, JavaScript, HTML
